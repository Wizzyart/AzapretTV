# Azapret TV

Экспериментальный Android TV APK с локальным VPN-сервисом и сетевыми профилями для выбранных приложений.

Идея продукта:

```text
Скачал APK -> записал на флешку -> установил на Android TV -> нажал Включить -> YouTube работает
```

## Статус

Приложение содержит:

- Android TV проект;
- главный экран, удобный для пульта;
- кнопку включения/выключения;
- Android `VpnService`;
- foreground notification;
- автозапуск после перезагрузки;
- TPWS/Orbit-профили;
- DPI-профили на базе локального прокси;
- ручной подбор профиля;
- базовую конфигурацию видеодоменов.

VPN работает локально на устройстве и не отправляет трафик на внешние серверы приложения.

## Почему APK Лучше Для Пользователя

Роутерный вариант требует модель, ревизию и иногда прошивку OpenWrt. Это рискованно для массового пользователя.

APK-вариант проще:

- не нужно знать модель роутера;
- не нужно прошивать устройство;
- не ломается домашний интернет;
- можно установить с флешки;
- можно обновлять как обычное приложение.

## Сборка

Открыть папку `azapret-tv` в Android Studio и собрать `debug` APK.

Минимальные требования:

- Android Studio;
- Android Gradle Plugin 8.x;
- Android SDK 35;
- Kotlin 2.x.

Команда при наличии Gradle/Android SDK:

```sh
./gradlew assembleDebug
```

## Установка На Android TV

1. Соберите APK.
2. Запишите APK на флешку.
3. Подключите флешку к телевизору или приставке.
4. Разрешите установку из неизвестных источников.
5. Установите APK.
6. Откройте `Azapret TV`.
7. Нажмите `Включить обход`.
8. Подтвердите системное VPN-разрешение Android.

## Важно

Android показывает системное VPN-предупреждение для любого приложения на базе `VpnService`. Это нормально: без этого приложение не может перехватывать трафик локально.
